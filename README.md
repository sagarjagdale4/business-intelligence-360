## Overview
An end-to-end business intelligence solution that consolidates **Finance, Sales, Supply Chain, and Executive** views. The dashboard demonstrates star-schema modeling, DAX measures (market share, RC%, GM%, Forecast accuracy), and interactive storytelling to support decision-making.

**Objective:** Help business users quickly identify profitability issues, forecast accuracy problems, and top/bottom performing products & customers.

---

## Key Features
- Multi-page report: Executive → Finance → Sales → Supply Chain → Support  
- P&L breakdown (Net Sales, COGS, Gross Margin, Net Profit)  
- Market share & RC% calculations (DAX)  
- Forecast Accuracy & Absolute Error analytics  
- Interactive slicers (Region, Segment, Product, Customer)  
- Clean visuals & exportable PDF

## How I built it
1. Data ingestion and cleaning using SQL + Power Query  
2. Star schema modeling (fact & dimension tables)  
3. Key DAX measures: `Total Sales`, `Market Share`, `RC%`, `Gross Margin %`, `Forecast Accuracy`  
4. Design & visualization: focused on interpretability and actionable insights
